This responsive visualization, built with web components, changes the visual representation of the data depending on the size of the window. All three are the same website rendering the same data. The last one displays each data point as a circle. The middle representation skips the rendering of individual data points, but instead uses the data-group to represent all nested data with a single line element. The first visualization is for mobile, as it flips the axes and displays the data as bars.
